# Quadratic Functions Explorer – Version 2
Δυναμική προσομοίωση στο GeoGebra για την εξερεύνηση της δευτεροβάθμιας συνάρτησης  
**f(x) = ax² + bx + c**

Η προσομοίωση επιτρέπει στους μαθητές να μεταβάλλουν τις παραμέτρους της παραβολής και να παρατηρούν άμεσα τις αλλαγές στο γράφημα, στην κορυφή, στις ρίζες, στη διάκριση και στη μορφή κορυφής.

---

## 🎯 Στόχοι μάθησης
- Κατανόηση της επίδρασης των παραμέτρων **a**, **b**, **c** στη μορφή της παραβολής.
- Διερεύνηση της **κορυφής**, του **άξονα συμμετρίας**, των **ριζών** και της **διάκρισης**.
- Σύνδεση της **τυπικής μορφής** με τη **μορφή κορυφής**.
- Ενίσχυση της οπτικοποίησης και της διερευνητικής μάθησης στα Μαθηματικά.

---

## 🧩 Χαρακτηριστικά της προσομοίωσης
- Sliders για τις παραμέτρους **a**, **b**, **c**.
- Αυτόματος υπολογισμός:
  - Κορυφής (h, k)
  - Άξονα συμμετρίας
  - Ριζών (όταν υπάρχουν)
  - Διάκρισης Δ
- Σημείο τομής με τον άξονα y.
- Εναλλαγή μεταξύ:
  - Τυπικής μορφής: *f(x) = ax² + bx + c*
  - Μορφής κορυφής: *f(x) = a(x − h)² + k*
- Κουμπί **Reset** για επαναφορά των τιμών.
- Δυναμικές ετικέτες και καθαρή οπτική διάταξη.
